# MCP Steroid

<p align="center">
  <img src="docs/pluginIcon.svg" alt="MCP Steroid Logo" width="120" height="120">
</p>

**An MCP Server that brings all IntelliJ-based IDE features to AI Agents, just the same way what IntelliJ brings to developers.**

MCP Steroid is an MCP (Model Context Protocol) plugin for IntelliJ IDEA that provides a Kotlin console interface, allowing LLM agents to execute code directly within the IDE's runtime environment. It makes coding Agents such as Claude Code, Codex, Gemini, Cursor, Junie, and OpenCode much more productive and efficient.
