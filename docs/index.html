About the Plugin

MCP Steroid is an MCP (Model Context Protocol) plugin for IntelliJ IDEA that provides a Kotlin console interface, allowing LLM agents to execute code directly within the IDE's runtime environment.

This IntelliJ plugin exposes IDE APIs to LLM agents via Kotlin script execution. Code runs in IntelliJ's own classpath with access to all installed plugins.

  • Access project structure and PSI (Program Structure Interface)
  • Use refactoring tools and VFS (Virtual File System)
  • Execute any IDE APIs directly from your AI agent
  • Built-in code review mode for safe execution
  • Demo mode overlay for presentations and debugging

Works with popular AI coding agents:

Claude Code Codex CLI Gemini CLI Cursor Junie OpenCode
